# Universal Conjoint Analysis Engine

This Streamlit application performs dynamic Conjoint Analysis.

## Features
- User-defined attribute structure
- Rating & Choice-based conjoint
- Automatic one-hot encoding
- Attribute importance calculation
- AI-based recommendation
- GitHub + Streamlit deployment ready

## How to Run Locally

pip install -r requirements.txt  
streamlit run app.py

## Deployment
Upload all files to GitHub and deploy via Streamlit Cloud.
